# S3 — Simple Storage Service

## Description
Scripts for managing S3 buckets: create, list, and delete.

## Prerequisites
- AWS credentials configured (`aws configure` or environment variables)
- `pip install -r requirements.txt`

## Scripts

### create_bucket.py
Create one or more S3 buckets with a random suffix.

```bash
python services/s3/create_bucket.py --prefix mybucket --count 3 --region us-west-2
python services/s3/create_bucket.py --prefix mybucket --profile dev
```

### list_buckets.py
List all S3 buckets in the account.

```bash
python services/s3/list_buckets.py
python services/s3/list_buckets.py --profile prod
```

**Expected output:**
```
2024-01-01T10:00:00 [INFO] services.s3.list_buckets: Bucket: mybucket12345
2024-01-01T10:00:00 [INFO] services.s3.list_buckets: Bucket: mybucket67890
```

### delete_bucket.py
Delete an S3 bucket by name. Use `--dry-run` to preview.

```bash
python services/s3/delete_bucket.py --name mybucket12345
python services/s3/delete_bucket.py --name mybucket12345 --dry-run
```
